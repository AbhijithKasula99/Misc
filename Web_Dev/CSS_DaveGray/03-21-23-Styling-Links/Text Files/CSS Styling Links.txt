a link it turns to purple, if you want it to change to another color we can use "a:visited" and use that to alter the code.

- Pseudo code >>>>> Normal code in terms of specificity. (Use the specificity calculator to check)

- Color theory and clealiness of the code is really important to maintain the standard of the code.

CSS code

@import url('https://fonts.googleapis.com/css2?family=Oxygen&display=swap');

body {
    font-family: 'Oxygen', sans-serif;
    padding: 10%;
    font-size: 2rem;
}

a { /* Normal element */
    text-decoration: underline;
    cursor: pointer;
    color: black;
}

a:visited { /* Pseudo element */
    color: purple;
}

a:hover {
    color: white;
    background-color: black;
    opacity: 0.7;
}

a:active {
    color: red;
}
