Scrape Kasula's Resume

I have left a download option to download my resume at the bottom, by clicking that my resume will be downloaded in .pdf format.

As this page is a html site, there is a possibility to scrape out the data available in this page, I have linked the resume in pdf format so if you had to get the file using web scraping, this is how you would do it.

I have chosen Requests package in python to parse this site as python is dynamic and the code is easy to understand for non-technical background people.

Scrape

It is better to use Jupyter Notebook and I have added the code snippet, that can be used to download my resume.

Use the copy button to copy the code snippet and paste it in your notebook! 

      import requests

      # specify the file URL
      url = 'http://127.0.0.1:5501/project-2/Resume_Abhijith_Kasula.pdf'
      
      # send a GET request to the URL and save the response content to a file
      response = requests.get(url)
      if response.status_code == 200:
          filename = 'Resume.pdf'
          with open(filename, 'wb') as f:
              f.write(response.content)
          print(f"File '{filename}' downloaded successfully!")
      else:
          print("Failed to download file!")

If the output statement states that the file has been downloaded sucessfully then the resume will be available in the same directory where the jupyter notebook file is present.
